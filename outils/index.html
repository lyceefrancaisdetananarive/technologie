🖨 Impression et export PDF

Chaque fiche d'activité et évaluation dispose d'un bouton "Imprimer / PDF" qui ouvre le dialogue d'impression de votre navigateur.

Conseils pour un bon rendu PDF
  • Utilisez Chrome ou Edge pour le meilleur rendu
  • Dans les options d'impression, choisissez "Enregistrer au format PDF" comme destination
  • Le format est optimisé pour du A4
  • L'en-tête LFT et les numéros de page apparaissent automatiquement
  • Les éléments de navigation sont masqués à l'impression
