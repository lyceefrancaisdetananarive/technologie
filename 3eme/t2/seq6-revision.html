Tests et mise au point — Les notions clés

❓ Quiz 📘 Structuration

📖 Vocabulaire essentiel

MotDéfinition
Protocole de testProcédure détaillée décrivant les conditions, étapes et critères d'un test.
Cahier de recetteDocument qui liste tous les tests à effectuer pour valider un produit.
Test fonctionnelVérification que l'objet remplit bien sa fonction attendue.
Test de performanceMesure des capacités de l'objet (vitesse, précision, autonomie).
ItérationCycle amélioration : tester → analyser → corriger → retester.

💡 Ce qu'il faut retenir

Démarche de test : Définir les critères → Rédiger le protocole → Exécuter les tests → Analyser les résultats → Améliorer.
  • Un protocole de test doit être reproductible : mêmes conditions = mêmes résultats.
  • On distingue test fonctionnel (« ça marche ? ») et test de performance (« ça marche bien ? »).
  • L'itération permet d'améliorer progressivement le produit à chaque cycle de test.
  • Les résultats doivent être consignés dans un tableau de mesures pour être analysés.
