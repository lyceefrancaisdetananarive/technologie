Séquence 4 — Systèmes automatisés complexes

Décomposition fonctionnelle

Un système complexe = plusieurs sous-systèmes fonctionnels, chacun avec sa propre chaîne fonctionnelle : capteur → traitement → actionneur.

Le microcontrôleur centralise le traitement de tous les sous-systèmes.

Séquence 5 — Programmation avancée

Opérateurs logiques

  • ET : les DEUX conditions doivent être vraies
  • OU : au moins UNE condition doit être vraie
  • NON : inverse la condition (VRAI ↔ FAUX)

Tableau de vérité

ABA ET BA OU B
VVVV
VFFV
FVFV
FFFF

Structures de programmation avancées

  • Variable : stocke une valeur modifiable (ex : tours ← 0)
  • Conditions imbriquées : SI...SINON SI...SINON (plusieurs cas)
  • Boucle TANT QUE : répète tant qu'une condition est vraie

Séquence 6 — Tests et mise au point

Protocole de test (5 étapes)

  1. Définir les critères de réussite
  2. Planifier les scénarios
  3. Exécuter les tests
  4. Observer les résultats
  5. Analyser les écarts

📝 Conseils pour l'évaluation

  • Savoir décomposer un système en sous-systèmes (capteur/traitement/actionneur)
  • Savoir dessiner un organigramme avec les bons symboles
  • Savoir utiliser les opérateurs logiques ET, OU, NON
  • Savoir écrire un pseudo-code avec conditions imbriquées et variables
  • Savoir identifier les 4 types d'erreurs et proposer des corrections
  • Savoir rédiger un protocole de test
