📖 Vocabulaire essentiel

MotDéfinition
PythonLangage de programmation textuel, simple et puissant, très utilisé en technologie.
FonctionBloc de code nommé et réutilisable qui effectue une tâche spécifique.
Liste (tableau)Structure de données qui stocke plusieurs valeurs dans une seule variable.
DébogageProcessus de recherche et correction des erreurs (bugs) dans un programme.
BibliothèqueEnsemble de fonctions prêtes à l'emploi importables dans un programme (ex : math, random).

💡 Ce qu'il faut retenir

En Python : variables, conditions (if/elif/else), boucles (for/while), fonctions (def) et listes sont les outils fondamentaux.
  • Une fonction se définit avec def nom(): et se réutilise autant de fois que nécessaire.
  • Les listes permettent de stocker et manipuler plusieurs données (notes = [12, 15, 8]).
  • Le débogage se fait étape par étape : lire l'erreur, localiser la ligne, corriger, retester.
  • Les bibliothèques évitent de tout recoder (ex : import random pour des nombres aléatoires).
