Activité 1 — IA embarquée vs IA cloud

1

Où se trouve l'intelligence ?

Compare deux approches d'intégration de l'IA dans les objets.

Deux types d'IA dans les objets :
  • IA embarquée (edge AI) : le traitement est réalisé directement dans l'objet (puce dédiée). Réponse rapide, fonctionne hors ligne.
  • IA cloud : les données sont envoyées à un serveur distant qui effectue le traitement. Plus puissant mais nécessite internet.
  • IA hybride : combinaison des deux (traitement local + affinage cloud).

Activité 2 — Étude de cas : la voiture autonome

2

Percevoir, Décider, Agir

Analyse le fonctionnement de l'IA dans une voiture autonome selon le modèle PDA.

Les 5 niveaux d'autonomie (SAE) :
  • Niveau 0 : Aucune aide (conducteur contrôle tout)
  • Niveau 1 : Assistance (ex : régulateur adaptatif)
  • Niveau 2 : Automatisation partielle (ex : Tesla Autopilot)
  • Niveau 3 : Automatisation conditionnelle (conducteur en backup)
  • Niveau 4 : Haute automatisation (zones définies, sans conducteur)
  • Niveau 5 : Automatisation complète (tout terrain, tout contexte)
