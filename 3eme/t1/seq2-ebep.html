💡 Le problème à résoudre

Comment l'IA fonctionne-t-elle dans les objets du quotidien ?

L'IA embarquée = une intelligence artificielle qui fonctionne directement dans l'objet, sans avoir besoin d'Internet.
Exemples : voiture autonome, robot aspirateur, smartphone (reconnaissance faciale).

Activité 1 — IA dans le cloud ou embarquée ?

Aide :
☁️ IA dans le cloud = les données sont envoyées sur Internet pour être traitées (ex : ChatGPT)
📱 IA embarquée = le traitement se fait dans l'objet même, sans Internet (ex : Face ID)

Coche la bonne colonne :

ExempleCloud ☁️Embarquée 📱
Alexa répond à une question⬜⬜
Face ID déverrouille le téléphone⬜⬜
Robot aspirateur qui contourne les meubles⬜⬜
Traduction automatique Google⬜⬜

Activité 2 — Comment l'IA de la voiture autonome « voit » ?

Aide : La voiture autonome utilise le modèle PDA :
👁️ Percevoir = voir (caméras, capteurs LiDAR)
🧠 Décider = réfléchir (l'IA analyse)
🚗 Agir = conduire (tourner, freiner, accélérer)

Complète :

Percevoir
La voiture « voit » avec des c................. et des capteurs
Décider
L'IA d................. s'il faut freiner ou continuer
Agir
La voiture f................. ou accélère automatiquement
