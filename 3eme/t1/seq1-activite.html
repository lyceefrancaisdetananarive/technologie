Activité 2 — Architecture et flux de données

2

Comment les données circulent dans un objet connecté

Analyse le parcours des données, de la captation à l'action.

Architecture type d'un objet connecté :
  1. Capteur : mesure une grandeur physique (température, mouvement, son…)
  2. Microcontrôleur : traite les données localement
  3. Module de communication : Wi-Fi, Bluetooth, Zigbee, 4G…
  4. Cloud / Serveur : stockage, analyse IA, historiques
  5. Application : interface utilisateur (smartphone, web)
  6. Actionneur : agit sur l'environnement (relais, moteur, LED…)
