📌 Schéma de câblage

Le schéma de câblage montre les connexions physiques entre les composants :

RègleExplication
AlimentationFil rouge = +5V ou +3.3V, fil noir = GND (masse)
SignauxChaque capteur/actionneur est connecté à une broche spécifique
Résistances de protectionToujours une résistance en série avec une LED (~220Ω)
Pull-up / Pull-downRésistance pour stabiliser un signal numérique (bouton)
À retenir
Un schéma de câblage est indispensable avant tout montage. Il permet de vérifier les connexions, éviter les courts-circuits et faciliter le débogage matériel.

📌 Planification du projet

Le projet Smart Object suit une démarche rigoureuse :

  1. Analyse du besoin et rédaction du CdC
  2. Conception : schéma de câblage + organigramme + modèle 3D du boîtier
  3. Prototypage matériel : montage sur breadboard
  4. Programmation : codage et débogage
  5. Fabrication du boîtier : impression 3D ou découpe laser
  6. Tests : protocole de test conforme au CdC
  7. Présentation : soutenance orale + démonstration

📚 Vocabulaire essentiel

Mots clés de la séquence
  • Smart Object : objet connecté capable de capter, traiter et communiquer des données
  • Schéma de câblage : représentation des connexions physiques entre composants
  • Breadboard : plaque de prototypage sans soudure pour tester les montages
  • ESP32 : microcontrôleur avec Wi-Fi et Bluetooth intégrés
  • MQTT : protocole de communication léger pour l'IoT
  • Pull-up / Pull-down : résistance stabilisant un signal numérique
