Objets connectés (IoT) — Les notions clés

❓ Quiz 📘 Structuration

📖 Vocabulaire essentiel

MotDéfinition
IoT (Internet of Things)Réseau d'objets physiques connectés qui échangent des données via internet.
ProtocoleRègle de communication entre appareils (Wi-Fi, Bluetooth, Zigbee, LoRa).
CloudServeurs distants qui stockent et traitent les données des objets connectés.
Passerelle (gateway)Dispositif qui relie le réseau local d'objets à internet.
Données personnellesInformations qui permettent d'identifier une personne (nom, localisation, habitudes).

💡 Ce qu'il faut retenir

Architecture IoT : Objets (capteurs) → Réseau (protocoles) → Cloud (stockage/traitement) → Application (interface utilisateur).
  • Un objet connecté combine capteurs, microcontrôleur et module de communication.
  • Les protocoles varient selon la portée : Bluetooth (courte), Wi-Fi (moyenne), LoRa (longue).
  • Le cloud centralise les données et permet l'accès à distance.
  • La protection des données personnelles est un enjeu majeur (RGPD en Europe).
