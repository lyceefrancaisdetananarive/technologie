Exercice 2 — Analyse et écriture de programme (12 points)

Compétence : C2

Contexte : On programme un système d'alarme : un capteur ultrason détecte une présence à moins de 30 cm. Si une présence est détectée, le buzzer sonne et la LED rouge s'allume. Sinon, tout reste éteint.

2.1 Complète le programme Arduino ci-dessous avec les instructions manquantes. (6 pts)

int distance;

void setup() {
  pinMode(9, ________); // Trig en sortie
  pinMode(10, ________); // Echo en entrée
  pinMode(7, ________); // LED en sortie
  pinMode(8, ________); // Buzzer en sortie
}

void loop() {
  // Mesure de distance (code fourni)
  digitalWrite(9, LOW); delayMicroseconds(2);
  digitalWrite(9, HIGH); delayMicroseconds(10);
  digitalWrite(9, LOW);
  distance = pulseIn(10, HIGH) / 58;

  if (________ < ________) { // Si distance < 30
    digitalWrite(7, ________); // Allumer LED
    ________; // Activer buzzer
  } else {
    digitalWrite(7, ________); // Éteindre LED
    ________; // Désactiver buzzer
  }
  delay(200);
}
