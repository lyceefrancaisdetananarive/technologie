✅ Éléments de correction — Activité 1

Points de vigilance FabLab :

  • Vérifier que les fichiers STL sont « étanches » (pas de trous dans le maillage) avant le slicing
  • Paramètres slicer recommandés : remplissage 20%, épaisseur couche 0.2 mm, supports si nécessaire
  • Pour la découpe laser xTool M1 : vérifier que le fichier SVG a les bonnes dimensions et que les traits de coupe sont en rouge (puissance 100%)
  • Temps d'impression moyen : 1h-2h pour un petit boîtier. Prévoir de lancer les impressions en avance si possible

Problèmes fréquents :

ProblèmeSolution
Pièce qui ne colle pas au plateauAjouter une bordure (brim) ou du scotch de peintre
Dimensions incorrectesVérifier l'échelle dans le slicer (1:1)
Pièces qui ne s'emboîtent pasPrévoir un jeu de 0.2-0.3 mm dans le modèle 3D
Fil qui casse pendant l'impressionVérifier que la bobine se déroule librement

✅ Éléments de correction — Activité 2

Ordre d'assemblage recommandé :

  1. Fixer l'Arduino dans le boîtier (vis M3 ou colle chaude)
  2. Positionner et fixer les capteurs aux ouvertures prévues
  3. Câbler les composants (vérifier les branchements AVANT d'alimenter)
  4. Téléverser le programme
  5. Tester chaque fonction séparément, puis le programme complet
  6. Fermer le boîtier

Erreurs courantes des élèves :

  • Inversion de polarité sur la LED (patte longue = +)
  • Fils mal enfoncés dans la breadboard → connexion intermittente
  • Oubli de la résistance en série avec la LED (risque de griller)
  • Mauvaise pin dans le programme (ne correspond pas au câblage)
