🎯 Exercice 1 — La carte Arduino

Consigne : Relie chaque élément de la carte Arduino à son rôle.
Élément→Rôle
1. Microcontrôleur A. Envoyer le programme depuis l'ordinateur
2. Port USB B. Lire des valeurs de 0 à 1023
3. Pins numériques C. Le « cerveau » qui exécute le programme

🎯 Exercice 3 — Compléter un programme

Consigne : Complète le programme pour faire clignoter une LED. Utilise les mots : HIGH, LOW, 1000.
void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, ________); // Allumer
  delay(________); // Attendre 1 seconde
  digitalWrite(13, ________); // Éteindre
  delay(1000);
}
💡 Aide : HIGH = allumer (5V), LOW = éteindre (0V), 1000 millisecondes = 1 seconde.

🎯 Exercice 4 — Comprendre une condition

Consigne : Lis le programme et réponds aux questions.
if (distance < 20) {
  digitalWrite(7, HIGH); // Allumer LED
} else {
  digitalWrite(7, LOW); // Éteindre LED
}
  1. Si la distance vaut 10 cm, la LED est-elle allumée ou éteinte ?  
  2. Si la distance vaut 25 cm, la LED est-elle allumée ou éteinte ?  
  3. Que faut-il changer pour que la LED s'allume quand la distance est inférieure à 30 cm ?  
💡 Aide : « if (distance < 20) » veut dire « SI la distance est plus petite que 20 ».
