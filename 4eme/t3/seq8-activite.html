Étape 3 — Écrire le programme « Blink »

Ouvre Arduino IDE et écris le programme suivant :

void setup() {
  pinMode(13, OUTPUT); // Pin 13 en sortie
}

void loop() {
  digitalWrite(13, HIGH); // Allumer la LED
  delay(1000); // Attendre 1 seconde
  digitalWrite(13, LOW); // Éteindre la LED
  delay(1000); // Attendre 1 seconde
}

Questions :

  1. Que fait la fonction setup() ?  
  2. Que fait la fonction loop() ?  
  3. Modifie le programme pour faire clignoter la LED 2 fois plus vite. Quelle valeur changes-tu ?

Étape 2 — Programme : mesurer une distance

int distance;

void setup() {
  pinMode(9, OUTPUT); // Trig
  pinMode(10, INPUT); // Echo
  Serial.begin(9600);
}

void loop() {
  digitalWrite(9, LOW);
  delayMicroseconds(2);
  digitalWrite(9, HIGH);
  delayMicroseconds(10);
  digitalWrite(9, LOW);
  distance = pulseIn(10, HIGH) / 58;
  Serial.print("Distance : ");
  Serial.print(distance);
  Serial.println(" cm");
  delay(500);
}
