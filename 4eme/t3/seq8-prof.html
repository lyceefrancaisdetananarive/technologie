✅ Éléments de correction — Activité 1

Identification des éléments Arduino :

ÉlémentRôle
Microcontrôleur (ATmega328)Le « cerveau » qui exécute le programme
Port USBAlimentation + communication avec l'ordinateur (téléversement)
Pins numériques (0-13)Entrées/sorties tout-ou-rien (0 ou 1) — boutons, LEDs
Pins analogiques (A0-A5)Lecture de valeurs 0 à 1023 — capteurs de lumière, température
Alimentation (5V, 3.3V, GND)Fournir l'énergie aux composants externes

Questions :

  1. setup() = initialisation, s'exécute une seule fois au démarrage (configuration des pins)
  2. loop() = boucle principale, s'exécute en continu indéfiniment
  3. Pour clignoter 2× plus vite : remplacer delay(1000) par delay(500)

✅ Éléments de correction — Activité 2

Condition if/else attendue :

if (distance < 15) {
  digitalWrite(ledPin, HIGH); // Allumer LED
} else {
  digitalWrite(ledPin, LOW); // Éteindre LED
}

Points de vigilance :

  • Vérifier que les élèves utilisent le bon sens du capteur (Trig = OUTPUT, Echo = INPUT)
  • Le moniteur série (Serial Monitor) doit être à 9600 bauds
  • Bien câbler la résistance en série avec la LED (risque de griller la LED sans résistance)

✅ Éléments de correction — Activité 3

Exemple de programme pour distributeur de gel :

#include <Servo.h>
Servo monServo;
int distance;

void setup() {
  pinMode(9, OUTPUT);  // Trig
  pinMode(10, INPUT);  // Echo
  monServo.attach(6);  // Servo pin 6
  monServo.write(0);   // Position initiale
}

void loop() {
  digitalWrite(9, LOW);
  delayMicroseconds(2);
  digitalWrite(9, HIGH);
  delayMicroseconds(10);
  digitalWrite(9, LOW);
  distance = pulseIn(10, HIGH) / 58;

  if (distance < 10) {
    monServo.write(90);  // Appuyer
    delay(1000);
    monServo.write(0);   // Relâcher
    delay(2000);         // Anti-spam
  }
  delay(200);
}
