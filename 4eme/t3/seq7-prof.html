✅ Éléments de correction — Activité 1

Rôles dans l'équipe :

  • Chef de projet : coordonne, vérifie le respect du planning, anime les réunions
  • Dessinateur : réalise les croquis et la modélisation 3D du boîtier
  • Programmeur : code le programme Arduino (capteurs → traitement → actionneurs)
  • Fabricant : prépare les fichiers (STL, SVG), lance les fabrications, assemble
