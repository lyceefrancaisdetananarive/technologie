Exercice 2 — Programmation Arduino (7 points)

Compétence : C2

Contexte : La station météo utilise un capteur de température (DHT11) branché sur la pin A0. Si la température dépasse 30°C, un ventilateur (pin 6) se met en marche et une LED rouge (pin 7) s'allume. Sinon, le ventilateur s'arrête et une LED verte (pin 8) s'allume.

2.1 Complète le programme Arduino. (4 pts)

int temperature;

void setup() {
  pinMode(6, ________); // Ventilateur
  pinMode(7, ________); // LED rouge
  pinMode(8, ________); // LED verte
}

void loop() {
  temperature = analogRead(A0);

  if (temperature > ________) {
    digitalWrite(6, ________); // Ventilateur ON
    digitalWrite(7, ________); // LED rouge ON
    digitalWrite(8, LOW);
  } else {
    digitalWrite(6, ________); // Ventilateur OFF
    digitalWrite(7, LOW);
    digitalWrite(8, ________); // LED verte ON
  }
  delay(1000);
}
