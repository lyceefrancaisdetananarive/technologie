📌 La carte Arduino

L'Arduino est un microcontrôleur programmable utilisé pour piloter des capteurs et des actionneurs :

ComposantRôle
Microcontrôleur (ATmega)Le « cerveau » qui exécute le programme
Entrées numériques (D0-D13)Lire des signaux ON/OFF (bouton, capteur de présence)
Entrées analogiques (A0-A5)Lire des valeurs continues (0-1023) — capteur de lumière, température
Sorties numériquesEnvoyer du courant ON/OFF — allumer LED, piloter relais
Sorties PWM (~)Simuler une sortie analogique — varier vitesse moteur, luminosité LED
Port USBConnecter au PC pour programmer et alimenter
À retenir
L'Arduino lit des entrées (capteurs) et pilote des sorties (actionneurs) grâce à un programme. Il fait le lien entre le monde physique et le programme informatique.

À retenir
Un programme Arduino suit toujours la structure : setup() (exécuté une fois au démarrage) + loop() (exécuté en boucle infinie). mBlock traduit les blocs visuels en code Arduino.

📌 De mBlock au code Arduino

mBlock permet de programmer en blocs visuels puis de voir le code Arduino correspondant :

  • Avantage des blocs : pas de syntaxe à mémoriser, idéal pour débuter
  • Avantage du code : plus de contrôle, plus de fonctionnalités, indispensable en lycée
  • Téléversement : le programme est envoyé dans la mémoire de l'Arduino via USB
  • Moniteur série : outil pour afficher les valeurs des capteurs sur l'écran du PC
À retenir
mBlock est un outil de transition entre la programmation par blocs (Scratch) et la programmation textuelle (C/C++ Arduino). Les deux produisent le même résultat.

📚 Vocabulaire essentiel

Mots clés de la séquence
  • Arduino : carte microcontrôleur programmable pour piloter capteurs et actionneurs
  • Capteur : composant qui mesure une grandeur physique (entrée)
  • Actionneur : composant qui produit une action physique (sortie)
  • PWM : modulation de largeur d'impulsion — simule un signal analogique
  • setup() : fonction exécutée une seule fois au démarrage de l'Arduino
  • loop() : fonction exécutée en boucle infinie après le setup
  • Téléversement : envoi du programme dans la mémoire de l'Arduino
  • Moniteur série : outil de débogage affichant les données sur le PC
