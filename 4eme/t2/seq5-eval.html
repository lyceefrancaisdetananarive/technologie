Exercice 2 — Programmation (12 points)

Compétence : C2

2.1 Analyse ce programme et réponds aux questions. (6 pts)

int capteur = A0;
int led = 13;

void setup() {
  pinMode(led, OUTPUT);
}

void loop() {
  int valeur = analogRead(capteur);
  if (valeur > 500) {
    digitalWrite(led, HIGH);
  } else {
    digitalWrite(led, LOW);
  }
}

a) Que fait ce programme ? (2 pts)

b) Que signifie analogRead(capteur) ? (1 pt)

 

c) Que se passe-t-il si la valeur du capteur est 300 ? (1 pt)

 

d) Que signifient HIGH et LOW pour la LED ? (2 pts)
