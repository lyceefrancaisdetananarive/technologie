Programmation Arduino / mBlock — Les notions clés

❓ Quiz 📘 Structuration

📖 Vocabulaire essentiel

MotDéfinition
ArduinoCarte électronique programmable utilisée pour créer des systèmes interactifs.
Entrée / SortieBroches de l'Arduino : entrée (capteur) ou sortie (LED, moteur, buzzer).
VariableEspace mémoire nommé qui stocke une valeur pouvant changer (ex : température).
FonctionBloc de code réutilisable qui effectue une tâche précise.
mBlockLogiciel de programmation par blocs compatible Arduino, basé sur Scratch.

💡 Ce qu'il faut retenir

Structure d'un programme Arduino : setup() (initialisation, exécuté 1 fois) + loop() (boucle principale, exécuté en continu).
  • Les capteurs sont branchés sur les entrées (analogiques ou numériques).
  • Les actionneurs sont branchés sur les sorties (LED, servomoteur, buzzer).
  • On utilise des variables pour stocker les valeurs lues par les capteurs.
  • Les conditions et boucles contrôlent le comportement du programme.
