Programmation et automatisation (mBlock / Scratch) — Les notions clés

❓ Quiz 📘 Structuration

📖 Vocabulaire essentiel

MotDéfinition
mBlockLogiciel de programmation par blocs basé sur Scratch, compatible Arduino.
ScratchEnvironnement de programmation visuelle par blocs visuels.
SpriteObjet graphique programmable (feu tricolore, bouton piéton...).
CostumeApparence d'un sprite (ex : 3 costumes = feu vert, orange, rouge).
MessageSignal envoyé entre sprites pour les synchroniser.
Interface graphiqueProgramme visuel permettant l'interaction utilisateur-système.

💡 Ce qu'il faut retenir

Cycle du feu : Vert (5 s) → Orange (2 s) → Rouge (5 s) → recommence.
  • On programme par blocs visuels : événements, contrôle, apparence, variables.
  • Un sprite a plusieurs costumes pour changer d'apparence (3 états du feu).
  • Les messages synchronisent les sprites (Feu 1 vert = Feu 2 rouge).
  • Une variable peut stocker si le bouton piéton a été appuyé (0 ou 1).
  • Les 2 feux d'un carrefour ne doivent jamais être verts en même temps.
