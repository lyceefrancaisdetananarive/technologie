Les formats de fichiers 3D

Un modèle 3D est un fichier numérique qui décrit la forme d'un objet dans l'espace (longueur, largeur, hauteur).

Formats de fichiers courants :
  • STL : format standard pour l'impression 3D (maillage de triangles)
  • 3MF : format amélioré pour l'impression 3D (couleurs, matériaux)
  • OBJ : format universel pour la 3D (textures, couleurs)

Activité 3 — Du modèle à la fabrication (40 min)

3

Préparer un fichier pour la fabrication

Avant de lancer la fabrication, il faut paramétrer le fichier dans un logiciel de tranchage (slicer) pour l'impression 3D, ou dans le logiciel de la xTool pour la découpe laser.

Étapes pour imprimer en 3D :
  1. Modéliser l'objet (ou télécharger un fichier STL/3MF)
  2. Importer dans le slicer (ex : Cura, PrusaSlicer)
  3. Paramétrer : épaisseur de couche, remplissage, supports
  4. Trancher et exporter le G-code
  5. Lancer l'impression
