📌 De l'algorithme au programme

ÉtapeCe que c'estQui le fait
1. AlgorithmeSuite d'instructions en langage courant pour résoudre un problèmeL'humain réfléchit
2. ProgrammeTraduction de l'algorithme dans un langage compréhensible par la machineL'humain code (blocs ou texte)
3. ExécutionLa machine lit et exécute les instructions une par uneLe microcontrôleur agit
À retenir
Un algorithme est une recette de cuisine pour la machine : des étapes précises, dans un ordre logique, pour obtenir un résultat.

📌 Les structures de base d'un programme

StructureRôleExemple en pseudo-code
SéquenceInstructions exécutées l'une après l'autreAllumer LED → Attendre 1s → Éteindre LED
Condition
(SI...ALORS...SINON)		Exécuter des instructions seulement si une condition est vraieSI bouton appuyé ALORS allumer LED SINON éteindre LED
Boucle
(RÉPÉTER / TANT QUE)		Répéter des instructions plusieurs foisRÉPÉTER 10 fois : faire clignoter LED
À retenir
Tout programme est construit avec seulement 3 structures : la séquence, la condition (SI) et la boucle (RÉPÉTER).

📌 Le micro:bit et mBlock

Le micro:bit est un microcontrôleur utilisé au collège pour apprendre la programmation :

Composant intégréCe qu'il mesure / fait
Matrice LED 5×5Afficher des icônes, texte, animations
Boutons A et BDétecter une pression (entrée utilisateur)
AccéléromètreDétecter les mouvements et l'inclinaison
BoussoleMesurer l'orientation (nord magnétique)
Capteur de luminositéMesurer la luminosité ambiante
Capteur de températureMesurer la température

mBlock est un logiciel de programmation par blocs (comme Scratch) compatible avec le micro:bit et l'Arduino.

📌 Les variables

Une variable est une boîte qui stocke une valeur qu'on peut lire et modifier.

  • Créer une variable : lui donner un nom (ex : score)
  • Affecter une valeur : score ← 0 (mettre 0 dans la boîte)
  • Modifier : score ← score + 1 (ajouter 1 au contenu)
  • Lire : afficher ou utiliser la valeur dans une condition
À retenir
Une variable est comme une étiquette sur une boîte : on lui donne un nom, on y met une valeur, et on peut la changer à tout moment.

📚 Vocabulaire essentiel

Mots clés de la séquence
  • Algorithme : suite ordonnée d'instructions pour résoudre un problème
  • Programme : algorithme traduit dans un langage compris par la machine
  • Boucle : structure qui répète des instructions (RÉPÉTER, TANT QUE)
  • Condition : structure qui teste si quelque chose est vrai ou faux (SI...ALORS...SINON)
  • Variable : espace mémoire nommé qui stocke une valeur modifiable
  • Capteur : composant qui mesure une grandeur physique
  • Microcontrôleur : petit ordinateur programmable (micro:bit, Arduino)
