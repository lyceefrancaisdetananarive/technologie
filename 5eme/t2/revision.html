Séquence 4 — Structure interne des objets techniques

📖 Vocabulaire essentiel

MotDéfinition
ComposantÉlément qui constitue un objet technique (ex : moteur, roue, batterie).
Fonction techniqueLe rôle d'un composant dans l'objet (ex : le moteur convertit l'énergie).
MatériauSubstance utilisée pour fabriquer un composant (aluminium, plastique...).
Chaîne d'énergieParcours de l'énergie dans un objet : Source → Convertir → Transmettre → Action.

💡 Les 4 familles de matériaux

FamilleExemplesPropriétés
MétauxAcier, aluminium, cuivreRésistants, conducteurs, recyclables
PlastiquesABS, PVC, caoutchoucLégers, isolants, moulables
Céramiques / verresVerre, porcelaine, bétonDurs, fragiles, isolants
CompositesFibre de carbone, fibre de verreTrès légers ET très résistants

Séquence 5 — Transmission et transformation de mouvement

📖 Vocabulaire essentiel

MotDéfinition
TransmissionLe mouvement d'entrée et de sortie sont du même type (rotation → rotation).
TransformationLe mouvement d'entrée et de sortie sont de types différents (rotation → translation).
EngrenageDeux roues dentées en contact qui transmettent un mouvement de rotation.
Chaîne d'informationParcours de l'information : Acquérir → Traiter → Communiquer.
CapteurComposant qui capte une grandeur physique (température, mouvement, lumière...).

💡 Mécanismes à connaître

MécanismeTypeExemple
EngrenagesTransmissionDérailleur de vélo
CourroieTransmissionMachine à laver
ChaîneTransmissionVélo
Bielle-manivelleTransformationMoteur de voiture
CrémaillèreTransformationPortail coulissant
Vis-écrouTransformationÉtau, cric

Séquence 6 — Initiation à la programmation

📖 Vocabulaire essentiel

MotDéfinition
ProgrammeSuite d'instructions exécutées par une machine dans l'ordre.
SéquenceInstructions exécutées les unes après les autres.
BoucleStructure qui permet de répéter des instructions.
ConditionStructure SI...ALORS qui permet de faire un choix.
Variable« Boîte » qui stocke une valeur modifiable (ex : score = 0).
Micro:bitPetit ordinateur programmable avec LEDs, boutons et capteurs.

💡 À retenir

  • Les 3 structures de base : séquence, boucle, condition
  • La programmation par blocs (mBlock) est visuelle et accessible
  • Le micro:bit possède : 25 LEDs, 2 boutons, accéléromètre, capteur température/lumière
  • Débugger = trouver et corriger les erreurs d'un programme

📝 Méthode — Pour bien répondre aux questions

Type de questionComment répondre
"Identifie les composants" Nommer le composant + sa fonction technique (ce qu'il fait dans l'objet).
"Classe ce matériau" Donner la famille (métal, plastique, céramique, composite) + une propriété.
"Décris la chaîne d'énergie" Source → Convertir → Transmettre → Action. Nommer chaque élément.
"Transmission ou transformation ?" Regarder si le mouvement change de type : même type = transmission, type différent = transformation.
"Écris un algorithme" Utiliser les mots-clés : QUAND, RÉPÉTER, SI...ALORS, variable ← valeur. Écrire chaque instruction sur une ligne.
