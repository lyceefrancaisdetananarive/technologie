📌 Les familles de matériaux

Tout objet technique est fabriqué à partir de matériaux choisis selon leurs propriétés.

FamilleExemplesPropriétés principales
MétauxAcier, aluminium, cuivreRésistants, conducteurs, recyclables
Plastiques / OrganiquesPLA, ABS, PET, boisLégers, isolants, moulables
CéramiquesVerre, porcelaine, bétonDurs, résistants à la chaleur, fragiles
CompositesFibre de carbone, fibre de verreTrès résistants et légers (combinent 2 matériaux)
À retenir
Le choix d'un matériau dépend de ses propriétés (masse, résistance, conductivité) et de l'usage prévu (légèreté pour un vélo, isolation pour un boîtier).

📌 La chaîne d'énergie

Pour fonctionner, tout objet technique a besoin d'énergie. La chaîne d'énergie décrit le parcours de l'énergie dans l'objet :

FonctionRôleExemple (trottinette électrique)
AlimenterFournir l'énergieBatterie lithium
DistribuerRéguler l'énergieContrôleur électronique
ConvertirTransformer l'énergieMoteur (électrique → mécanique)
TransmettreAmener l'énergie à l'actionCourroie + roue
À retenir
Alimenter → Distribuer → Convertir → Transmettre : les 4 fonctions de la chaîne d'énergie, toujours dans cet ordre.

📌 La chaîne d'information

La chaîne d'information décrit comment un objet technique perçoit son environnement et décide quoi faire :

FonctionRôleComposant type
AcquérirCapter une grandeur physiqueCapteurs (température, distance, lumière)
TraiterAnalyser et déciderMicrocontrôleur (Arduino, micro:bit)
CommuniquerAfficher ou transmettre le résultatÉcran LCD, LED, buzzer, Wi-Fi
À retenir
Acquérir → Traiter → Communiquer : la chaîne d'information pilote la chaîne d'énergie. Le capteur détecte, le microcontrôleur décide, l'actionneur agit.

📌 Représentations techniques

  • Nomenclature : tableau listant tous les composants d'un objet (repère, nom, quantité, matériau)
  • Vue éclatée : dessin montrant les pièces séparées mais dans l'ordre d'assemblage
  • Vue en coupe : dessin montrant l'intérieur d'un objet (comme si on le coupait)

📚 Vocabulaire essentiel

Mots clés de la séquence
  • Matériau : substance dont est fait un objet (métal, plastique, céramique, composite)
  • Chaîne d'énergie : parcours de l'énergie dans un objet (alimenter → distribuer → convertir → transmettre)
  • Chaîne d'information : parcours de l'information (acquérir → traiter → communiquer)
  • Capteur : composant qui mesure une grandeur physique (température, distance, lumière)
  • Actionneur : composant qui agit sur l'environnement (moteur, LED, buzzer)
  • Nomenclature : tableau qui liste tous les composants d'un objet technique
