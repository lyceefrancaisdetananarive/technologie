💡 Le problème à résoudre

Comment dessiner un objet en 3D sur un ordinateur ?

Avant de fabriquer un objet, on le dessine sur un ordinateur.
On utilise un logiciel de modélisation 3D (comme Tinkercad).
Le modèle 3D montre l'objet sous tous les angles, comme dans la vraie vie.

Activité 1 — Les vues d'un objet

Aide : Pour représenter un objet en 2D (sur papier), on utilise 3 vues :
👁️ Vue de face = ce qu'on voit devant
👁️ Vue de dessus = ce qu'on voit d'en haut
👁️ Vue de côté = ce qu'on voit sur le côté
