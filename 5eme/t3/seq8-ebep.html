💡 Le problème à résoudre

Comment travailler en équipe pour fabriquer un objet technique ?

Un projet de groupe = plusieurs élèves travaillent ensemble pour créer un objet.
Il faut s'organiser, se répartir les tâches et communiquer.

Activité 1 — Les étapes d'un projet

Aide : Un projet suit toujours des étapes dans l'ordre. C'est la démarche de projet.

Numérote les étapes de 1 à 5 :

Étape Ordre
🏗️ Fabriquer le prototype
🔍 Comprendre le besoin
✅ Tester et améliorer
📝 Rédiger le cahier des charges
✏️ Dessiner la solution (maquette)

Activité 2 — Mon rôle dans l'équipe

Aide : Dans un projet, chaque membre de l'équipe a un rôle : chef de projet, dessinateur, fabricant, testeur.

Complète ta fiche d'équipe :

Nom du projet
Mon nom
Mon rôle
Ma tâche cette semaine
