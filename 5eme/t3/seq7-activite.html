💡 Situation déclenchante

Concevoir avant de fabriquer
Avant de construire un pont, une maison ou même un simple support de téléphone, les ingénieurs créent un modèle 3D numérique. Cela permet de vérifier les dimensions, tester la solidité et corriger les erreurs avant la fabrication.

Problématique : Comment utiliser la modélisation 3D pour concevoir un objet technique avant de le fabriquer ?

Activité 1 — Le cahier des charges

1

Comprendre le cahier des charges

Avant de concevoir un objet, on rédige un cahier des charges qui liste les contraintes à respecter.

Le cahier des charges contient :
  • La fonction d'usage : à quoi sert l'objet ?
  • Les contraintes techniques : dimensions, poids, matériaux...
  • Les contraintes esthétiques : forme, couleur, design...
  • Les contraintes économiques : coût de fabrication
  • Les contraintes environnementales : recyclable, éco-conçu...

Activité 2 — Découverte de Tinkercad

2

Prends en main l'outil de modélisation 3D

Tinkercad est un logiciel en ligne gratuit qui permet de créer des objets 3D en assemblant des formes simples.

Les outils de base de Tinkercad :
  • Formes primitives : cube, cylindre, sphère, cône, tore...
  • Déplacer : cliquer-glisser une forme sur le plan
  • Redimensionner : tirer les poignées blanches
  • Regrouper : fusionner plusieurs formes en une seule
  • Percer : utiliser une forme "trou" pour creuser une autre forme
  • Aligner : centrer des formes les unes par rapport aux autres

Activité 3 — Concevoir le support de smartphone

3

Modélise ton support de smartphone sur Tinkercad

En respectant le cahier des charges de l'activité 1, conçois un support de smartphone fonctionnel et esthétique.

Critères d'évaluation du modèle 3D :
  • Le support tient debout de manière stable
  • Le téléphone est maintenu sans tomber (angle d'inclinaison adapté)
  • Les dimensions sont réalistes et respectent le cahier des charges
  • Le design est soigné (formes harmonieuses, nom gravé...)
