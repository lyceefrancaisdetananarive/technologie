Modélisation 3D — Les notions clés

❓ Quiz 📘 Structuration

📖 Vocabulaire essentiel

MotDéfinition
Modélisation 3DReprésentation numérique d'un objet en trois dimensions (longueur, largeur, hauteur).
CAOConception Assistée par Ordinateur : logiciel pour dessiner des pièces en 3D.
VolumeForme pleine dans l'espace : cube, cylindre, sphère, etc.
EsquisseDessin 2D de base utilisé pour créer un volume 3D (extrusion, révolution).
Impression 3DFabrication d'un objet physique couche par couche à partir d'un modèle numérique.

💡 Ce qu'il faut retenir

Processus de modélisation : Esquisse 2D → Volume 3D → Modifications (trous, chanfreins) → Export pour fabrication.
  • Un logiciel de CAO (TinkerCAD, BlocksCAD, OnShape) permet de concevoir des objets en 3D.
  • On part toujours d'une esquisse 2D que l'on transforme en volume (extrusion).
  • Les cotes (dimensions) doivent être précises pour que l'objet soit fabriquable.
  • Le fichier 3D est exporté en STL pour l'impression 3D.
